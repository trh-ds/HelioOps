# Role & Context
Act as a world-class, award-winning Senior Frontend Developer and UI/UX Designer. 
Your task is to build a premium, highly realistic, and breathtaking landing page for "HelioOps" — a cutting-edge Space Weather Impact Intelligence platform that predicts GPS L1 Errors and HF Radio Blackouts using advanced Machine Learning models.

# Tech Stack & Guidelines
- Framework: Next.js (React), Vue, or pure HTML/CSS/JS (use whatever you are most confident in to deliver the best result).
- Styling: Tailwind CSS for utility-first, highly responsive styling.
- Animations & Scrolling: Framer Motion, GSAP, and Lenis (or Locomotive Scroll) for silky smooth, premium scrolling effects. Consider Three.js/React-Three-Fiber for realistic 3D space backgrounds.
- Quality: Ensure the code is production-ready, modular, and visually flawless.

# Design Aesthetics
- Theme: Deep Space / Sci-Fi / High-Tech Military Dashboard. Dark mode is mandatory.
- Colors: Deep void blacks (`#050505`), starfield dark grays, accented with vibrant, energetic colors representing space weather phenomena: Solar Flare Orange/Red, Aurora Borealis Neon Green (`#00ff88`), and Ultraviolet.
- Typography: Modern, sleek sans-serif fonts (e.g., 'Inter', 'Space Grotesk') for headings. Monospaced fonts (e.g., 'Roboto Mono', 'JetBrains Mono') for data readouts, coordinates, and metrics.
- UI Elements: Heavy use of "Glassmorphism" (frosted glass, background blur) for overlay cards. Subtle inner glows, 1px semi-transparent borders, and drop shadows to create depth. It must feel like a premium scientific instrument.

# Page Structure & Scrolling Effects (The "Story")
The landing page should guide the user through a cinematic story as they scroll, using advanced scroll-triggered animations.

1. Hero Section (The Void):
- Visual: A hyper-realistic, slow-moving background of the Sun emitting a solar flare, or Earth shielded by a glowing magnetic field.
- Content: The title "HelioOps" in bold, glowing, futuristic typography. Subtitle: "Predictive Space Weather Impact Intelligence." 
- Effect: Intense Parallax scrolling. As the user scrolls down, the background elements move at different speeds. The hero text should elegantly fade and slide up on initial load.

2. The Threat (Scroll Section 1):
- Content: Explaining the catastrophic impact of Coronal Mass Ejections (CMEs) and Solar Storms on global infrastructure (Aviation, Military, GPS).
- Effect: "Sticky" or "Pinned" scrolling section. The user scrolls, but the main container stays pinned while text panels slide in from the side or bottom, and the background visual transitions from the Sun to an impacted Earth.

3. The Intelligence - ML Engine (Scroll Section 2):
- Content: Highlight the AI prediction capabilities. Mention "GPS L1 Error Tracking" and "HF Radio Blackout Probability" with verified 95% confidence intervals and an R² score of 0.98.
- Effect: Animate data cards popping in dynamically (staggered reveal) as they enter the viewport. Implement number counters that dynamically count up from 0 to "98.5%" or "95%" when visible.

4. Real-time Dashboard Preview (Scroll Section 3):
- Visual: A stunning, glassmorphic mock-up of the HelioOps dashboard showing glowing charts, gauges for Kp Index, and CME speeds.
- Effect: 3D Tilt effect on hover. Cards should rotate slightly based on mouse position to give a tangible, physical feel.

5. Call to Action / Footer:
- Content: "Deploy HelioOps for your infrastructure." A sleek, glowing CTA button (e.g., "Request Access" or "Initialize Radar").
- Effect: The button should have a continuous subtle pulse or sweeping light ray animation.

# Specific Implementation Rules
- NEVER use generic, flat placeholder colors (like standard red or blue). Always use carefully curated gradients and hex codes.
- Prioritize micro-interactions: Buttons that glow on hover, links that reveal underlines smoothly, and cards that subtly lift.
- DO NOT just provide a basic template. Strive for a "WOW" factor that would win an Awwwards Site of the Day.
- Output the complete, structured code. If using React/Next.js, break it down logically by components.

DELIVERABLE: Generate the complete code for this frontend. Focus heavily on the visual aesthetics, the realistic space theme, and the advanced scroll-linked animations.
